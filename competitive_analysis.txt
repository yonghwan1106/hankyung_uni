2020-2024년 제출 아이디어 대비 우위 분석
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【 분석 목적 】

HK투명 제안이 과거 5년간(2020-2024) 제출된 대학발전 아이디어와
어떻게 차별화되는지 명확히 하여, 심사위원에게 "새로운" 제안임을
설득력 있게 제시함.

【 구체적 차별화 분석 】

1. 일회성 vs 지속성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

과거 전형적 제안:
"2025년 OO 페스티벌 개최"
"△△ 특강 시리즈 (10회)"
"□□ 장학생 선발 프로그램"

→ 예산 소진 후 종료
→ 매년 새로 기획·실행 필요
→ 누적 효과 제한적

HK투명:
"실시간 투명성 플랫폼 구축"

→ 1회 구축 후 영구 운영
→ 매일 24시간 작동
→ 시간이 지날수록 데이터 축적, 가치 증가
→ 10년 후에도 여전히 유효


2. 특정 집단 vs 전체 시민
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

과거 전형적 제안:
"외국인 학생 지원 프로그램" → 수혜자: 50명
"지역 고등학생 진로 체험" → 수혜자: 300명
"창업 동아리 지원" → 수혜자: 5개 팀

→ 명확한 수혜 집단
→ 타겟팅은 명확하나 범위 제한
→ "왜 우리는 혜택을 못 받나?" 형평성 논란 가능

HK투명:
"전 국민 접근 가능한 투명성 플랫폼"

→ 수혜자: 무제한 (인터넷 접속 가능한 모든 시민)
→ 안성시민, 평택시민, 전국 납세자 모두 혜택
→ 형평성 논란 없음 (모두에게 개방)
→ 네트워크 효과: 사용자 많을수록 가치 증가


3. 비용 증가형 vs 비용 절감형
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

과거 전형적 제안:
"신규 실습실 구축" → 초기 5억 + 연간 유지 5천만
"셔틀버스 노선 확대" → 연간 운영비 2억
"외부 전문가 초청 강연" → 연 10회 × 500만원 = 5천만

→ 지속적 예산 투입 필요
→ 인플레이션 따라 비용 증가
→ 예산 부족 시 프로그램 중단 위험

HK투명:
1차년도: 2천만원
2차년도 이후: 연 500만원

→ 초기 투자 후 운영비 최소화
→ 오히려 행정 효율화로 간접 비용 절감
  (정보공개청구 응대 시간 ↓, 민원 처리 ↓)
→ 디지털 자산은 감가상각 없음
→ 장기적으로 가장 경제적


4. 내부 성과 vs 외부 가시성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

과거 전형적 제안:
"학사행정 시스템 개선"
"교수 연구 지원 확대"
"내부 회의 효율화"

→ 실제로 중요하고 필요한 개선
→ 그러나 외부(시민, 언론)에 보이지 않음
→ 홍보 효과 제한적
→ "국민이 신뢰하는 대학" 테마와 간접 연결

HK투명:
"실시간 공개 투명성 플랫폼"

→ 외부에 즉시, 명확하게 보임
→ 언론 보도 가치 극대화
  ("전국 최초" 뉴스가치 높음)
→ SNS 확산 가능성 (시민이 자발적 공유)
→ "국민이 신뢰하는 대학" 테마에 직접 정렬


5. 수동 보고 vs 자동 공개
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

과거 전형적 제안:
"연간 백서 발간"
"분기별 뉴스레터 발송"
"홈페이지 공지사항 게시"

→ 담당자가 수동으로 작성·업로드
→ 시간차 발생 (실시간 아님)
→ 선택적 공개 가능성
→ 지속 노력 필요

HK투명:
"실시간 자동 데이터 연동"

→ 기존 시스템 API 연결로 자동 업데이트
→ 조작·지연 불가능 (시스템 자동화)
→ 24/7 실시간 공개
→ 인력 부담 최소


6. 단일 주제 vs 통합 플랫폼
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

과거 전형적 제안:
각각 별개의 제안:
• "학생 취업률 향상 프로그램"
• "연구비 투명성 강화"
• "지역사회 기여 확대"
• "캠퍼스 친환경화"

→ 개별 문제에 개별 솔루션
→ 통합 시각 부족
→ 시너지 효과 제한

HK투명:
하나의 플랫폼에서 모든 영역 통합:
• 예산 투명성
• 연구 성과
• 학생 성공
• 지역 기여
• 환경 성과

→ 종합적(holistic) 접근
→ 데이터 간 연계 분석 가능
→ 시민이 한 곳에서 모든 정보 확인
→ "신뢰"라는 테마를 전방위로 구현
