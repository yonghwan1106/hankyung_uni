# HK투명 - 실시간 투명성 플랫폼 데모

한경국립대학교 대학발전 아이디어 제안서의 3대 핵심 시스템을 실제 구현한 인터랙티브 데모입니다.

## 🎯 구현된 시스템

### 시스템 A: 실시간 투명성 대시보드
- ✅ **예산 투명성**: 월별/분야별 예산 집행 현황 (54억원, 48.9% 집행률)
- ✅ **연구 성과**: 논문/특허/기술이전 실적 (342편, 45건, 12건)
- ✅ **학생 성공**: 취업률/창업/장학금 현황 (78.5%, 47개, 1,847명)
- ✅ **지역 기여**: 프로그램/경제기여 분석 (156개, 128억원)
- ✅ **환경·ESG**: 에너지/폐기물/탄소배출 관리 (18.5% 재생에너지)
- ✅ **통합 KPI**: 종합 평가 및 ROI 분석 (2.37배 투자 수익)

### 시스템 B: 시민 참여 시스템
- ✅ **시민 투표**: 실시간 투표 + 결과 확인 (3,262명 참여)
- ✅ **시민 Q&A**: 질문·답변 게시판 (7일 이내 답변)
- ✅ **아이디어 제안**: 시민 제안 및 채택 시스템 (투표 + 검토)

### 시스템 C: 정기 공개 보고
- ✅ **월간 리포트**: 매월 첫 주 전월 실적 공개 (PDF 다운로드)
- ✅ **분기별 타운홀**: 총장 타운홀 미팅 (온·오프라인 + YouTube 생중계)
- ✅ **연간 성과보고서**: 종합 보고서 발행 (매년 1월, 전년도 성과)

## 🚀 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 열기

### 3. 프로덕션 빌드
```bash
npm run build
npm start
```

### 4. 정적 사이트 생성 (Vercel/Netlify 배포용)
```bash
npm run build
```

생성된 `out/` 폴더를 배포

## 📁 프로젝트 구조

```
│   │   ├── budget/         # 예산 대시보드
│   │   ├── research/       # 연구 성과

│   │   ├── environment/    # 환경 ESG
│   │   └── overview/       # 통합 KPI
│   ├── participate/        # 시스템 B: 시민 참여
│   │   ├── vote/           # 시민 투표
│   │   ├── qna/            # Q&A
│   │   └── ideas/          # 아이디어 제안
│   └── reports/            # 시스템 C: 정기 보고
│       ├── monthly/        # 월간 리포트

│       └── annual/         # 연간 보고서
├── components/

├── lib/
│   └── mockData.ts         # 목업 데이터
└── package.json
```

## 🎨 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React

## 📊 목업 데이터

모든 데이터는 `lib/mockData.ts`에서 관리됩니다:

- 예산 데이터 (월별 54억원 예산 집행)
- 연구 데이터 (342편 논문, 45건 특허)
- 학생 데이터 (78.5% 취업률, 47개 창업)
- 지역 기여 (156개 프로그램, 128억원 경제 기여)
- 환경 데이터 (18.5% 재생에너지, 69% 재활용)
- 투표 데이터 (3,262명 참여 실시간 투표)
- Q&A, 아이디어, 보고서 등

## 🎯 주요 기능

### 시스템 A: 투명성 대시보드

#### 예산 대시보드 (`/dashboard/budget`)
- 📊 월별 예산 집행 추이 (바 차트)
- 🥧 분야별 예산 배분 (파이 차트)
- 📈 집행률 추이 (라인 차트)
- 💰 주요 집행 내역 카드

#### 연구 대시보드 (`/dashboard/research`)
- 📈 논문 발표 및 피인용 추이 (라인 차트)
- 🥧 분야별 논문 분포 (파이 차트)
- 🏆 특허 출원/등록 현황
- 💼 기술이전 성과 및 수익

#### 학생 대시보드 (`/dashboard/students`)
- 📊 학과별 취업률 (바 차트)
- 🥧 산업별 취업 분포 (파이 차트)
- 🚀 학생 창업 현황 (47개 기업)
- 💰 장학금 지급 실적

#### 지역기여 대시보드 (`/dashboard/community`)
- 📊 프로그램 유형별 참여 인원 (바 차트)
- 🥧 프로그램 분류 (파이 차트)
- 💰 경제적 기여도 분석 (128억원)
- 🏢 시설 개방 및 이용 현황

#### 환경 대시보드 (`/dashboard/environment`)
- 📈 에너지 사용 및 탄소배출 추이 (라인 차트)
- 🥧 폐기물 유형별 배출량 (파이 차트)
- ♻️ 재활용률 및 감축 목표
- 🌱 재생에너지 비율 (18.5%)

#### 통합 KPI 대시보드 (`/dashboard/overview`)
- 🎯 종합 평가 레이더 차트 (투명성, 신뢰도, 만족도, 기여도)
- 📊 전년 대비 개선 현황 (바 차트)
- 💎 국민 투자 대비 경제적 가치 (ROI 2.37배)
- 📈 주요 성과 요약 카드

### 시스템 B: 시민 참여

#### 시민 투표 (`/participate/vote`)
- ✅ 실시간 투표 참여
- 📊 투표 결과 실시간 집계
- 👥 연령대별/지역별 참여 통계
- 📜 지난 투표 결과 아카이브

#### 시민 Q&A (`/participate/qna`)
- ❓ 질문 작성 및 카테고리 분류
- 💬 관련 부서 답변 (7일 이내)
- 👍 질문 추천 및 조회수
- 🔍 카테고리별 필터링

#### 아이디어 제안 (`/participate/ideas`)
- 💡 아이디어 제안 시스템
- 👍 커뮤니티 투표 및 추천
- ✅ 채택 아이디어 현황 (예산, 일정)
- 📊 상태별 필터링 (채택됨, 검토중, 논의중)

### 시스템 C: 정기 공개 보고

#### 월간 리포트 (`/reports/monthly`)
- 📄 매월 첫 주 전월 실적 공개
- 📊 예산/연구/취업/지역 요약
- 📥 PDF 다운로드
- 📧 이메일 구독 서비스

#### 분기별 타운홀 (`/reports/quarterly`)
- 🎤 총장 및 보직자 타운홀 미팅
- 📅 오프라인 참석 신청 (선착순 100명)
- 📺 YouTube 생중계 및 아카이브
- 💬 실시간 질의응답

#### 연간 보고서 (`/reports/annual`)
- 📊 전년도 종합 성과 발표
- 📈 전년 대비 개선 현황
- 💎 투자 대비 경제적 가치 (ROI)
- 📥 PDF 다운로드 및 알림 구독
