# HK투명 프로젝트 - 완성 보고서

## 📋 프로젝트 개요

**프로젝트명**: HK투명 - 실시간 투명성 플랫폼
**제안자**: 박용환 (크리에이티브 넥서스)
**제안 대상**: 한경국립대학교
