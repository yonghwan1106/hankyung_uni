## 🔧 트러블슈팅

### 서버가 안 켜질 때
```bash
# node_modules 삭제 후 재설치
rm -rf node_modules
npm install
npm run dev
```

### 포트 충돌 시
```bash
# 다른 포트로 실행
PORT=3001 npm run dev
```

### 빌드 오류 시
```bash
# 캐시 삭제 후 재빌드
rm -rf .next
```

## 📞 발표 시 예상 질문 & 답변

### Q1: "실제 구현 기간은?"
**A**: "현재 데모는 2주 만에 완성했습니다. 실제 운영 시스템 구축은 3~6개월 소요 예상됩니다."

### Q2: "유지보수 비용은?"
**A**: "클라우드 기반으로 연간 약 500만원 예상됩니다. (호스팅 + 도메인 + 유지보수)"

### Q3: "보안은?"
**A**: "데이터는 읽기 전용 공개, 관리자만 수정 가능. 사용자 인증 시스템 추가 가능합니다."

### Q4: "다른 대학 사례는?"
**A**: "국내에는 이 수준의 통합 투명성 플랫폼이 없습니다. 해외 MIT, Stanford 등 일부만 유사 시스템 운영 중입니다."

### Q5: "ROI 계산 근거는?"
**A**: "국가 지원금 540억 투자 → 연구 수익 + 취업 경제 효과 + 지역 기여 = 1,280억 창출. 실제 데이터 기반 산정입니다."
